Easiest -> Hardest skill implementation order

1) Attack - base damage (Damage event modifier)
2) Defence - damage reduction or block stamina cost
3) Strength - crit chance + crit damage
4) Ranged - damage + crit for ranged weapons
5) Magic - damage + crit for magic weapons
6) Architect - crafting refunds + extra outputs (furniture/decor/building blocks)
7) Cooking - extra outputs + longer food buffs
8) Alchemy - extra outputs + longer potion buffs
9) Mining - break speed + tool durability + yield
10) Woodcutting - break speed + tool durability + yield
11) Farming - planting gate + growth time + tool durability + yield
12) Smelting - processing bench pipeline
13) Leatherworking - processing bench pipeline
14) Arcane Engineering - RNG crafting system
15) Armorsmithing - RNG crafting system
16) Weaponsmithing - RNG crafting system
17) Fishing - custom subsystem
18) Slayer - custom subsystem
